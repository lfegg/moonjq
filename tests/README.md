# 测试目录说明

## 测试脚本

### 主要脚本

- **`test_all.ps1`** - 综合功能测试套件，测试所有核心功能
- **`examples_shell.ps1`** - Shell 命令风格示例演示
- **`examples_file.ps1`** - 文件读取功能示例演示

### 运行测试

```powershell
# 运行所有测试
.\tests\test_all.ps1

# 查看 Shell 风格示例
.\tests\examples_shell.ps1

# 查看文件读取示例
.\tests\examples_file.ps1
```

## 根目录脚本

- **`run.ps1`** - 快速启动 REPL 的便捷脚本
